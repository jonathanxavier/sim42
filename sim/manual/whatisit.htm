What is it?

 

Simulator42 is an open source process simulator project with the goal of providing an affordable and accessible simulator to the chemical engineering community.

Some interesting features of the project:

It is being written in the Python language

The simulator core has designed to be independent of both user interfaces and thermodynamic methods providers.

A graphical interface is being developed for the project and an independent command/scripting interface is also available. Other interfaces, including proprietary third party interfaces are expected.

The flow sheet solver can propagate partial information both backwards and forward through the flow sheet, which allows many complex problems to be solved without iterative calculation of recycle loops. Consistency checks are used to avoid unintended over specification of problems.

The simulator does not require recycle unit operations, but rather the use of estimate values allows the solver to recognize the existence of recycles and converge them simultaneously.

A distillation column employing a Russell inside/out algorithm and capable of solving complex pump around, side water draws and side stripper problems has been implemented.
