Property Value Status (bit maps)

 

These constants define the state of a value/

Variable

Value

Meaning

UNKNOWN_V

 

1

 

Value is not known

 

FIXED_V

 

2

 

Value was specified by user

 

CALCULATED_V

 

4

 

Value was calculated by a unit operation

 

PASSED_V

 

8

 

Value was propagated by the Sim42 solver

 

NEW_V

 

16

 

Value is newly specified

 

ESTIMATED_V

 

32

 

Value was estimated

 

PARENT_V

 

64

 

Value was calculated by a parent unit operation into to port of its child.
