Simulation Constants

 

These are commonly used constants for the definition of unit operation parameters.

Variable

Meaning

LiquidPhases

 

Number of liquid phases to be supported by port. The default is one

 

Trays

 

Number of trays in distillation tower

 

NumberStages

 

Number of trays in distillation tower

 

NumberStreamsIn

 

Number of material streams entering a unit operation.

 

NumberStreamsOut

 

Number of material streams leaving a unit operation

 

 

 

 

 

MaxNumIterations

 

Maximum number of iterations

 

RefluxRatio

 

Reflux ratio in distillation tower

 

SignalType

 

Type of signal being broadcast by sensor.

 

Ignored

 

If ignored is true unit operation will not solve.
