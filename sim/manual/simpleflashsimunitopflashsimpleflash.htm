SimpleFlash - sim.unitop.Flash.SimpleFlash

 

The separator is used to separate gas and liquid streams.

 

s = Flash.SimpleFlash()

 

Item

Value

Sim42 File

 

Flash.py

 

Example

 

Man-Sep2.tst

 

Man-Sep3.tst

 

In Material Port

 

Separator inlet

 

Vap Material Port

 

Separator vapor outlet

 

Liq0 Material Port

 

Separator liquid outlet. Note that the default for a separator is a single liquid outlet.

 

LiquidPhases

 

Tells the separator how many liquid phases it should consider. The default is 1

 

Equations

 

 

 

 

If you are running VMGSim, the SimpleFlash unit operation is equivalent to the Separator unit operation

 

#simple two phase separator example

thermo = VirtualMaterials.NRTL/Ideal/HC

thermo + NITROGEN OXYGEN ETHANOL WATER

 

s = Flash.SimpleFlash()

s.In.T = 25 C

s.In.P = 1 atm

s.In.MoleFlow = 100

s.In.Fraction = 1 1 1 1

 

s.Vap

s.Liq0

 

You can change the SimplaFlash behavior to handle three phase calculations as well. Let's modify the example above and add some toluene to it, thus creating three phases.

 

#simple two phase separator example

thermo = VirtualMaterials.NRTL/Ideal/HC

thermo + NITROGEN OXYGEN ETHANOL WATER TOLUENE

 

s = Flash.SimpleFlash()

s.In.T = 25 C

s.In.P = 1 atm

s.In.MoleFlow = 100

s.In.Fraction = 1 1 1 5 5

 

s.Vap

s.Liq0

Note that we have quite a bit of toluene and water in the liquid phase and there's a chance decantation is happening. Thus we change the LiquidPhases parameter from the default 1 to 2.

If you are using VirtualMaterials as your thermodynamic property engine three phase calculations are on by default. This means that the flash results are always rigorous even if you see only a recombined liquid phase.
