Sim42 Conceptual Structure Tour

 

In this section we will examine the basic structure of Sim42. Sim42 is a kernel, i.e., it provides dedicated simulation services such as simulation object creation, sequencing of information and convergence methods, but it does not provide native GUI services or any native input or output services. In this section we examine how the kernel fits into an overall software product framework.

A schematic view of Sim42’s structure is shown below.

Note that the command line interface (CLI) is a convenient, command driven interface to the raw API exposed by Sim42 (S42API). You may elect to create your own interfaces based on S42API or on CLI. You use CLI when you run the CommandInterface.py script. This interface resembles a DOS shell where you can type commands that are then interpreted by Sim42. As an example, VMGSim is a professional graphical user interface to Sim42 and it uses CLI to communicate with Sim42.
