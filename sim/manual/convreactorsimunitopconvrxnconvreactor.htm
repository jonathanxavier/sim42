ConvReactor - sim.unitop.ConvRxn.ConvReactor

 

The ConvReactor unit operation is designed to perform the necessary material and energy balances around a conversion reactor. A conversion reactor is a reactor where the user specifies the conversion of each reaction with respect to a key component. Each reaction is specified by defining the stoichiometric relationships between reactants and products.

 

rxn = ConvRxn.ConvReactor()

Equations

 

 

In order to define the reactor, we have to initially define the chemical reactions taking place. This is done using the Rxn<n>.Formula command. The syntax for the definition of a chemical reaction is:

Rxn0.Formula = PropaneAFT:4*WATER+3*"CARBON DIOXIDE"-!PROPANE-5*OXYGEN

•

The "PropaneAFT" is a tag for the reaction

•

The basis component is designated using the ! symbol

•

The numbers prefacing the component name are the stoichiometric coefficients for the chemical reaction

•

You can also use the position of the component instead of the component name in the Formula definition:

PropaneAFT:4*4+3*3-!0-5*1

•

Components with a blank in the name have to be written using the "" enclosing the name.

Useful tidbit

 

Combustion reactions for hydrocarbons can be written using the following general formula:

 

 

thermo = VirtualMaterials.Peng-Robinson

thermo + HYDROGEN WATER CARBON_MONOXIDE CARBON_DIOXIDE

thermo + METHANE OXYGEN NITROGEN AMMONIA ETHANE PROPANE
