Troubleshooting installation

 

Using CVS to get the source code is fairly easy. In windows, you can just download it to the folder where you wish to work, open your DOS prompt and type

cvs -d:pserver:anonymous@cvs.sim42.org/home/cvs login

then just press enter when asked for a password. To download everything now type

cvs -d:pserver:anonymous@cvs.sim42.org/home/cvs co sim

Most windows installations should work with out any extra work. The most common problem you'll encounter (usually Win98 or Win95) is an error related to a missing "home" folder. If you wan to get things going right away, just type SET HOME=<path> (where <path> is the path of where you are going to download sim42, for example D:). To avoid typing that line every time you want to download, you can go to your autoexec.bat file and add the same line there and restart your computer.

 

The most common problem you will face when running this application is "paths". This shouldn't be a problem if you keep certain things in mind:

§

Click "Yes" when asked about setting the system path while installing VMG software.

§

Make sure Python is also in your system path (should be done by default after installing it).

§

Most likely, your application of sim42 will not be in the Python path. The easiest way to work this out is to open your Notepad, write the path of your sim folder and save it as sim.pth in a valid python path location. For example, I have the folder sim in my D drive, so I wrote D:\ in the sim.pth file and saved it in D:\Python22\Lib\site-packages .
