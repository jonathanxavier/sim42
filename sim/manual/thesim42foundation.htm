The Sim42 Foundation

 

The Sim42 Software Foundation has been created for the following purposes:

To promote the development of an open source chemical engineering process simulator.

Towards this end the foundation will:

1.

Maintain servers that provide public access to the project source code, documentation, discussion lists and other tools that may be deemed appropriate.

2.

Decide what code and modifications are appropriate for inclusion in versions of the simulator bearing the names Sim42 or Simulator42.

3.

Be the copyright holder and license grantor for all code included in versions of the simulator bearing the names Sim42 or Simulator42.

4.

Take other actions deemed appropriate to further the principal objective.
