Set - sim.unitop.Set.Set

 

The set is used to specify a value based on a value calculated by a different port.

 

set = Set.Set()

Remember that you can only set variables that have the same unit.

 

 

You should be careful when using the multipliers for variables expressed in non absolute units. For example, if you are setting a temperature, and the system of units is not SI, you will get apparently unexpected results since Sim42 will convert the temperature to its absolute value and then apply the multiplier.

As a general rule, if you have a variable that may be expressed in non absolute terms leave the multiplier at 1.0
